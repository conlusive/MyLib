# MyLib 📚

**MyLib** is a Django-based web application for managing a digital library. It allows users to manage books, authors, and user accounts through a clean interface.

## 🔧 Technologies

- Python 3.x  
- Django 4.x  
- HTML/CSS (custom or Bootstrap)  
- SQLite (default database)
